-> Are the HTML tags and elements the same thing?
A) HTML tags and elements are related, but they are not exactly the same thing.

HTML tags are the markup symbols that are used to describe the structure of an HTML document. These tags typically come in pairs, with an opening tag and a closing tag, and they define how the content within them should be formatted or displayed. For example, the <p> tag is used to indicate a paragraph, and the <img> tag is used to indicate an image.

HTML elements, on the other hand, refer to the combination of an opening tag, content, and a closing tag. An HTML element is made up of the HTML tag, its attributes, and any content that is contained within the opening and closing tags. For example, the following is an HTML element that uses the <p> tag:

<p>This is a paragraph of text.</p>

In this example, the <p> tag is the HTML tag, and the opening and closing tags together with the text "This is a paragraph of text." form an HTML element.

So, while HTML tags are used to define the structure and formatting of an HTML document, HTML elements are the actual building blocks of the document, made up of the tags and their content.

-> What are HTML Entities? 
A) An HTML entity is a piece of text ("string") that begins with an ampersand ( & ) and ends with a semicolon ( ; ). Entities are frequently used to display reserved characters (which would otherwise be interpreted as HTML code), and invisible characters (like non-breaking spaces).

-> How can we club two or more rows or columns into a single row or column in an HTML table?
A) We use the colspan and rowspan attribute, to merge cells in HTML. The rowspan attribute is for the number of rows a cell should merge, whereas the colspan attribute is for the number of columns a cell should merge. The attribute should be placed inside the <td> tag.

-> What is SVG?
A) SVG stands for scalable vector graphics, which is a file type that permits resizing images without compromising on their quality. It’s an XML-based mark-up language that describes 2D vector graphics. SVG is a type of vector art. SVGs can be searched, indexed, scripted and compressed. This is easy because these types of images are based on mathematical shapes and curves rather than on pixels. SVGs can be animated, they support transparency, and support all types of colors and gradients.
